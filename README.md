# Avalanche-payments-foundry

Smart contract and payment splitter using Solidity on the Avalanche Fuji Testnet.  
This project demonstrates on-chain payments with Foundry, OpenZeppelin libraries, and deployment scripts.

---

## Foundry Toolkit

**Foundry is a blazing fast, portable and modular toolkit for Ethereum application development written in Rust.**

It consists of:
- **Forge**: Ethereum testing framework (like Truffle, Hardhat, DappTools).
- **Cast**: Swiss army knife for interacting with EVM smart contracts.
- **Anvil**: Local Ethereum node, akin to Ganache/Hardhat Network.
- **Chisel**: Fast Solidity REPL.
